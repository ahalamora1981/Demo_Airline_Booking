## 使用指南
1. 阅读“航班预订信息标准格式”，并输入您的预订信息。
2. 如果信息不完整，机器人将继续向您提问，直到获取完整的预订信息为止。
3. 确认航班预订信息标准格式内的全部信息，机器人将感谢您的预订并结束本次航班预订流程。
